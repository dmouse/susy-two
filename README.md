Susy two example
========

This is an example using Silex and Susy two 


Install
=========
```bash
$ git clone https://github.com/dmouse/susy-two.git 
$ cd susy-two
$ composer install

```

Development
========

```bash
$ cd web/css/Susy
$ compass compile
$ compass watch

```

Requirements
========

* Sass
* Compass
* Breakpoint
* Susy

```bash 
$ gem install sass compass breakpoint susy
```
